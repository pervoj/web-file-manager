# Web file manager

Simple file manager

## Instalation

1. Download latest release source code
2. Upload files (`index.php` and `wfm` directory) from downloadet zip to webhosting

## Update

1. Download latest release source code
2. Replace old files (`index.php` and `wfm` directory) with files from downloaded zip
